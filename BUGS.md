# BUG-01 
**Отрицательная цена допускается**

Шаги:
1. Отправить POST /api/1/item
2. Передать price = -100

Фактический результат:
- объявление успешно создается

Ожидаемый результат:
- ошибка 400
- сообщение о некорректной цене

Серьёзность: Major
Окружение: https://qa-internship.avito.com, POST /api/1/item

# BUG-02 

**В системе есть объявления с пустым name**

Шаги:
1. Получить объявления по sellerId
2. Проверить поле name

Фактический результат:
- в ответе есть объявления с пустым и другими name

Ожидаемый результат:
- name не должен быть пустым или отличающимся

Серьёзность: Medium 
Окружение: https://qa-internship.avito.com, POST /api/1/item
