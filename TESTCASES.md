## Версия 2 API

### 4. Удаление объявления (DELETE /api/2/item/{{itemID}})

**Tест 4.1. Delete тест (create → delete → verify)**

Создаlbv объявление через POST /api/1/item
Из ответа берёn item_id
Объявление удаляется через DELETE /api/2/item/{item_id}
Проверяеn, что объявление удалено через GET /api/1/item/{item_id}

Фактический результат::
- CREATE: 200 OK — объявление создано
- DELETE: 200 OK — объявление удалено
- GET after delete: 404 — объявление не найдено

Вывод:Удаление объекта проходит успешно, данные после удаления недоступны.

Окружение:
https://qa-internship.avito.com
