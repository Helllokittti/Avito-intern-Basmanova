## E2E сценарий

**Tест 6.1. E2E сценарий (создание, получение и удаление объявления)**

Сделано:
1. Отправим POST /api/1/item с валидными данными
2. Получили id созданного объявления
3. Отправили GET /api/1/item/{id} 
4. Создаю объявление через POST /api/1/item 
5. Из ответа берём item_id 
6. Объявление удаляется через DELETE /api/2/item/{item_id} 
7. Проверяем, что объявление удалено через GET /api/1/item/{item_id}


Ожидаемый результат:
- статус 200
- объявление найдено
- данные совпадают с отправленными
- объявление удалено
- объявление не найдено

Фактический результат:
- объявление успешно создается
- данные совпадают
- DELETE: 200 OK — объявление удалено
- GET after delete: 404 — объявление не найдено

Вывод:
- сценарий работает корректно
