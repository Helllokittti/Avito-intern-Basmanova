# BUG-01 
**Отрицательная цена допускается**

Шаги:
1. Отправить POST /api/1/item
2. Передать price = -100

Фактический результат:
- объявление успешно создается

Ожидаемый результат:
- ошибка 400
- сообщение о некорректной цене

Серьёзность: Minor
Окружение: https://qa-internship.avito.com, POST /api/1/item

# BUG-02 
**Некорректный тип поля status**

Шаги:
1. Отправить некорректный запрос (например без name)

Фактический результат:
- поле status возвращается как строка ("400")

Ожидаемый результат:
- поле status должно быть числом (400)

Серьёзность: Major
Окружение: https://qa-internship.avito.com, POST /api/1/item

# BUG-03 
**В системе есть объявления с пустым  name с несуществующей ценой**

Шаги:
1. Получить объявления по sellerId
2. Проверить поле name

Фактический результат:
- в ответе есть объявления с пустым и другими name

Ожидаемый результат:
- name не должен быть пустым или отличающимся

Серьёзность: Minor 
Окружение: https://qa-internship.avito.com, POST /api/1/item
